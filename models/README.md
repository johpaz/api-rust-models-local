# Modelos GGUF

Coloca aquí tus modelos GGUF descargados.

## Descarga Rápida

Usa el script incluido:
```bash
./scripts/download-model.sh bartowski/google_gemma-4-E4B-it-GGUF google_gemma-4-E4B-it-Q4_K_M.gguf
```
